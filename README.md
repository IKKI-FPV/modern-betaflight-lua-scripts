# Modern Betaflight Lua Scripts UI


ENGLISH

A complete restyling of the Betaflight Lua Script interface for EdgeTX radios with color screens and touch screens (e.g., Radiomaster TX16S, TX15, Boxer Max).

## üì• Installation
1.  Download the official Betaflight Lua scripts: [Betaflight Configurator Link](https://github.com/betaflight/betaflight-tx-lua-scripts)
2.  Copy the `BF` folder to your SD Card in `SCRIPTS/TOOLS/`.
3.  Download the files from the "Releases" section of this repository.
4.  Replace the original `ui.lua` and `ui_init.lua` files in the `SCRIPTS/TOOLS/BF/` folder with these modified ones.
5.  Launch the script from your radio.
